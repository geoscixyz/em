Understanding the Transient EM response

Purpose

We provide expressions for transient responses of the circuit model, compare them with harmonic responses, and understand their decay features.

For the harmonic response, we used normalized EMF or magnetic field:

\[\frac{H_3^s }{H_3^p} = \frac{\mathcal{E}_3^s }{\mathcal{E}_3^p} = - \frac{M_{12}M_{23}}{M_{13}L} \Big[\frac{\alpha^2 + \imath \alpha}{1+\alpha^2}\Big]\]

which was reduced to

\[\frac{H_3^s }{H_3^p} = \frac{\mathcal{E}_3^s }{\mathcal{E}_3^p} = C Q (\alpha)\]

where \(C\) and \(Q\) are coupling coefficient and response function, respectively. We did not consider effect of \(C\), but focus on \(Q\) because it determines time behavior in transient response.

However, for the transient response this neat normalization does not work, because of some complicated time function in \(\mathcal{E}_3^s\) and \(\mathcal{E}_3^p\) (assuming step-on current waveform):

\[\mathcal{E}^p_{on} = - M_{13} I_1 \delta(t)\]
\[\mathcal{E}^s_{on} = - \frac{M_{12}M_{23}}{L} \frac{I_1 u(t)}{\tau} e^{-t / \tau}\]

All terms in above EMFs have dimensions hence scale is much more important for the transient response compared to the harmonic one. In following, we work through time domain expressions for the current, response function, and EMF.

Current at the target loop

Current in the target loop for harmonic case is

\[\begin{split}I_2 e^{\imath \omega t} = - \frac{\imath \omega \tau}{ 1 + \imath \omega \tau} \frac{M_{12}I_1}{L} e^{\imath \omega t} \\\end{split}\]

Corresponding step-on response, \(i_{2 \ on}\) will be

\[i_{2 \ on} = \mathcal{F}^{-1}[\frac{I_2(\omega)}{\imath \omega}] = - \frac{M_{12}I_1}{L} e^{- t/\tau}, \ t>0\]

Using the relationship shown in Transient response, we can obtain step-off response, \(i_{2 \ off}\):

\[i_{2 \ off} = i_{2 \ on}(\infty) - i_{2 \ on}(t) = \frac{M_{12}I_1}{L} e^{- t/\tau}, \ t>0\]

Only sign is different between \(i_{2 \ on}\) and \(i_{2 \ off}\).

Impulse response, \(i_2(t)\) can be simply obtained by taking time derivative to \(i_{2 \ on}\):

\[i_2(t) = \frac{\partial i_{2 \ on}} {\partial t} = \frac{M_{12}I_1}{L} \frac{1}{\tau} e^{- t/\tau}, \ t>0\]

Response function

Response function, \(Q\) for the harmonic case is:

\[Q(\omega) = \frac{\imath \omega \tau}{1+\omega\tau}\]

Similarly, for step-on, -off, impulse responses can be written as

\[q_{on}(t) = e^{-t/\tau}, \ t>0\]
\[q_{off}(t) = -e^{-t/\tau}, \ t>0\]

and

\[q(t) = -\frac{1}{\tau} e^{-t/\tau}, \ t>0\]

Induced EMF at the Rx loop

For the harmonic case, primary and secondary EMFs at the receiver loop are:

\[\mathcal{E}^p (\omega) = - \imath \omega M_{13} I_1 e^{\imath \omega t}\]
\[\mathcal{E}^s (\omega) = - \imath \omega M_{23} I_2 e^{\imath \omega t} = \imath \omega \frac{M_{12}M_{23} I_1}{L} \Big[\frac{\imath \omega \tau}{ 1 + \imath \omega \tau} \Big] e^{\imath \omega t}\]

Similarly then step-on responses can be derived as

\[\mathcal{E}^p_{on}(t) = -M_{13} I_1 \delta(t)\]
\[\mathcal{E}^s_{on}(t) = \frac{M_{12}M_{23}}{L} \frac{I_1}{\tau} e^{-t / \tau}, \ t>0\]

Step-off responses are:

\[\mathcal{E}^p_{off}(t) = M_{13} I_1 \delta(t)\]
\[\mathcal{E}^s_{off}(t) = -\frac{M_{12}M_{23}}{L} \frac{I_1}{\tau} e^{-t / \tau}, \ t>0\]

Note

Both primary EMFs: \(\mathcal{E}^p_{on}\) and \(\mathcal{E}^p_{off}\) are zero when \(t>0\). Therefore, after zero time, we only measure secondary EMF.

We do not consider impulse response for EMF because it is not well-defined.
