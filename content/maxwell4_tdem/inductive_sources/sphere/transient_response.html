Transient Response

Purpose

The transient response of an object defines its step-off excitation; that is, the response by an object once a static field is removed. Here, we analyze the sphere’s transient response in terms of the induced magnetic dipole moment. The dependence of the transient response on the sphere’s physical properties are examined for permeable and non-permeable spheres.

Introduction

The field \(h_0(t)\) which induces a transient response from the sphere can be described using the following expression:

(439)\[h_0 (t) = H_0 u(-t)\]

where \(u(t)\) is the unit-step function and \(H_0\) is the amplitude of the applied field at \(t\leq 0\). From the previous section, the induced magnetic moment \(m(t)\) characterizing the sphere is defined by a convolution:

(440)\[m(t) = \Bigg ( \frac{4\pi}{3} R^3 \Bigg ) \int_{-\infty}^\infty \chi (\tau) h (t-\tau )d\tau\]

where \(\chi (t)\) represents the excitation’s impulse response. By substituting Eq. (439) into Eq. (440) we obtain the following:

(441)\[m(t) = \Bigg ( \frac{4\pi}{3} R^3 \Bigg ) \Bigg ( \int_{t}^\infty \chi (\tau) d\tau \Bigg ) H_0\]

From Eq. (441), the sphere’s transient response is linearly proportional to the strength on the inducing field. The characteristic behavior of the decay at time \(t\) is determine by integrating the impulse response from \(t\) to \(\infty\). Our analysis of the transient response is done by examining analytic solutions to Eq. (441) for permeable and non-permeable spheres.

Conductive Sphere

Here, we consider the transient response from a conductive and non-permeable (\(\mu = \mu_0\)) sphere. For a step-off excitation, the induced dipole moment for the sphere is given by:

(442)\[\begin{split}\begin{split} m(t) = \Bigg ( \frac{4\pi}{3}R^3 \Bigg ) \Bigg ( \frac{9}{2} \Bigg ) \Bigg [ \frac{1}{3} + \frac{t}{\beta^2} - \frac{2}{\beta} \sqrt{\frac{t}{\pi}} \Bigg ( 1 + 2 & \sum_{n=1}^\infty e^{-(n \beta)^2/t} \Bigg ) \; ... \\ &+ 4 \sum_{n=1}^\infty n \, \textrm{erfc} \Bigg ( \frac{n \beta}{\sqrt{t}} \Bigg ) \Bigg ] H_0 \, u(t) \end{split}\end{split}\]

where:

(443)\[\beta = (\mu_0 \sigma )^{1/2} R\]

and erfc(\(z\)) is the complementary error function given by:

(444)\[\textrm{erfc}(z) = \frac{2}{\sqrt{\pi}} \int_z^\infty e^{-t^2} dt\]

According to Eq. (442), the sphere does not experience an excitation while the field is applied; as electromagnetic induction is zero. At \(t=0\) however, the time-dependent change in the applied magnetic field induces eddy currents within the sphere. These eddy currents diffuse over time, dependent on the size and conductivity of the sphere. The strength of the induced current, and thus the strength of the sphere’s inductive response, is characterized by the sphere’s magnetic dipole moment.

The rate of decay for the sphere’s magnetic dipole moment can be obtained by taking the derivative of Eq. (442). For \(t>0\):

(445)\[\frac{d m}{dt} = \Bigg ( \frac{4\pi}{3}R^3 \Bigg ) \Bigg ( \frac{9}{2} \Bigg ) \Bigg [ \frac{1}{\beta^2} - \frac{1}{\beta \sqrt{\pi t}} \Bigg ( 1 + 2 \sum_{n=1}^\infty e^{-(n\beta)^2/t} \Bigg ) \Bigg ] H_0\]

As an example, let us consider a sphere of radius \(R=10\) m. The sphere is subject to a unit step-off excitation; thus \(H_0=1\). In Fig. 140, we show the transient response for several conductivities. We can see that for low conductivities, the strength of the induced dipole moment decays much more quickly. Since \(\beta\) increases as a function of \(R\) as well as \(\sigma\), we expect to observe a similar relationship in terms of the sphere’s radius. Additionally, we see that for early times, the rate of decay is proportional to \(t^{-1/2}\).

Conductive and Magnetically Permeable Sphere

Here, we consider the transient response from a conductive and non-permeable sphere. For a step-off excitation, the induced dipole moment for the sphere is given by:

(446)\[m(t) = \Bigg ( \frac{4\pi}{3}R^3 \Bigg ) \Bigg [ \frac{3}{2} \Bigg ( \frac{2 (\mu_r - 1)}{\mu_r + 2} u(-t) + 6 \mu_r \, \sum_{n=1}^\infty \frac{ e^{- \, \xi_n^2 t/\beta^2}}{(\mu_r + 2)(\mu_r - 1)+\xi_n^2} u(t) \Bigg ) \Bigg ] H_0\]

where \(\mu_r = \mu/\mu_0\) is the relative permeability of the sphere and:

(447)\[\beta = \big ( \mu \sigma \big )^{1/2} R\]

Coefficients \(\xi_n\) within the sum are defined by:

(448)\[\textrm{tan} \, \xi_n = \frac{(\mu_r - 1)\xi_n}{\mu_r - 1 + \xi_n^2}\]

From Wait and Spies ([WS69]), these coefficients are spaced roughly \(\pi\) apart such that:

(449)\[n\pi \leq \xi_n \leq (n+1/2) \pi\]

The value of each coefficient may be found iteratively using very few iterations (< 10) according to:

(450)\[\xi_n^{(k+1)} = n\pi + \textrm{tan}^{-1}\Bigg ( \frac{(\mu_r - 1) \xi_n^{(k)}}{\mu_r - 1 + (\xi_n^{(k)} )^2} \Bigg )\]

According to Eq. (446), the sphere’s response is separated into two terms; one for \(t\leq 0\) and one for \(t>0\). The first term, which is non-zero for \(t \leq 0\), represents the sphere’s steady-state magnetic response. As we can see from Eq. (446), this response depends only on the sphere’s magnetic properties. A detailed description of the steady-state magnetic response of a permeable sphere can be found here. The induced dipole moment at \(t>0\) represents the sphere’s inductive response. When comparing Eqs. (442) and (446), we see that the sphere’s transient response now depends on the magnetic permeability.

The rate of decay for the sphere’s magnetic dipole moment can be obtained by taking the derivative of Eq. (446). For \(t>0\):

(451)\[\frac{d m}{dt} = - \, \Bigg ( \frac{4\pi}{3}R^3 \Bigg ) \Bigg [ 9\mu_r \, \sum_{n=1}^\infty \frac{ \xi_n^2 \, e^{- \, \xi_n^2 t/\beta^2}}{\beta^2 \big [ (\mu_r + 2)(\mu_r - 1)+\xi_n^2 \big ]} \Bigg ] H_0\]

As an example, let us consider a sphere of radius \(R=10\) m. The sphere is subject to a unit step-off excitation; thus \(H_0=1\). In Fig. 141, we show the transient response for several relative permeabilities. For \(\mu_r < 6\), the strength of the inductive response increases with respect to the relative permeability but ultimately shows similar behaviour with respect to \(t\). For large relative permeabilities however, the characteristic decay of the induced dipole moment is altered by the sphere’s magnetic properties.
